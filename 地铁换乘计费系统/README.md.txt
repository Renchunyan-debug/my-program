# 地铁计价器（51单片机）

基于51单片机的地铁票价计算系统，支持A/B双线、跨线换乘计价。用户通过矩阵键盘输入起点和终点（线号+站号），系统自动计算票价并显示在LCD1602上。

## 硬件平台
- 51单片机（STC89C52RC）
- 4×4矩阵键盘
- LCD1602显示屏

## 主要功能
- 输入起点和终点（十位表示线路：1=A线，2=B线；个位表示站点1-9）
- 支持同线直通和跨线换乘两种计价模式
- 自动计算站点数，按规则计价
- LCD1602实时显示输入过程和最终票价
- 按键功能：数字键输入，11确认下一个，10开始计算，16清空

## 项目结构
code/
├── main.c # 主程序，计价逻辑与界面控制
├── MatrixKey.c/h # 4×4矩阵键盘驱动
├── LCD1602.c/h # LCD1602驱动（含各种显示函数）
└── Delay.c/h # 延时函数

## 遇到的问题及解决
- **问题**：跨线换乘时站点数计算容易出错  
  **解决**：分别计算起点到5号站、5号站到终点的站数，再相加。
- **问题**：输入界面不够直观  
  **解决**：增加提示文字（如“input two number”、“sure next empty”），引导用户操作。
