# 地铁计价器（51单片机）

基于51单片机的地铁票价计算系统，支持A/B双线、换乘计价。

## 硬件平台
- 51单片机（STC89C52RC）
- 4×4矩阵键盘
- LCD1602显示屏

## 主要功能
- 按键切换A线/B线
- 上下键选择站点（1-9）
- 确定站点并累计
- 自动计算票价（每3站5元，不足3站按5元计）
